# TL;DR

適合，但有條件：欄位控制在 3-4 個以內、人數在 5-8 人之間、且討論時要讓筆記「被看見」（例如投影對比）。結構本身不產生學習效果，結構讓比較變得可能，而比較才是學習發生的地方。

# 結論

**結構化筆記在小型讀書會中有效，但有效的原因不是「格式統一」，而是「讓差異可見」。**

三輪實驗顯示，精簡的三欄模板（一句話重點、一個疑問、一個連結）配合投影對比，讓參與者開始注意到彼此對同一段文字的理解差異。這種差異本身就是高品質討論的觸發點。

相對地，欄位太多的模板（五個以上）反而降低了參與度。成員把寫筆記當成「交作業」，失去了自發性。

**結構的甜蜜點是：夠少到不覺得是負擔，夠多到能形成可比較的維度。**

# 證據

**支持結構化的證據：**
- 三欄模板的完成率 85%，遠高於五欄模板的 30%
- 投影對比後，討論時長平均增加 40%，且討論內容從「我覺得這章講得不錯」轉向「為什麼你的重點跟我完全不同」
- 「一個連結」欄位迫使參與者做知識遷移，這與《Make It Stick》中的「生成效應」一致

**反對或保留的證據：**
- Roam Research 社群的經驗分享指出，過度結構化會抑制自由聯想
- Cornell 筆記法的研究主要針對個人使用，群體場景的效果可能來自不同機制
- 第一輪的失敗不只是欄位太多——也可能是成員還不理解結構化的目的

# 不確定性

- 目前的實驗只有三輪、共約六週的數據。長期使用後，成員會不會對固定模板產生疲勞？
- 8 人以上的讀書會，投影對比變得混亂。是否需要改用分組比較？
- 這個方法高度依賴「願意寫筆記」的參與者。如果讀書會的文化本來就不鼓勵書寫，導入模板可能適得其反
- 非虛構類書籍的效果明確，但文學或藝術類讀書會是否適用，完全沒有數據
